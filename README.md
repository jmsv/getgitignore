# gitignore-dl

[![npm version](https://badge.fury.io/js/gitignore-dl.svg)](https://badge.fury.io/js/gitignore-dl)

CLI tool to get a .gitignore file for your language from [GitHub's collection](https://github.com/github/gitignore)

```sh
npx gitignore-dl
```
